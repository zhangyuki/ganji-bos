### Usage

模板基于cooking构建，需要安装cooking

```
npm install cooking-cli -g

```


``` bash
# install dependencies
npm install

# dev
cooking watch -p

#build
cooking build

```

### Browser Support

Modern browsers and Internet Explorer 10+.
